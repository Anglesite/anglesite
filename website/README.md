# Pairadocs Farm

This is the source code for your farm website. It lives in iCloud Drive so it's backed up automatically.

## Getting started

Open this folder in Claude Desktop's Code tab, then type `/setup`. Claude will install everything and walk you through it.

## What you can do

| I want to… | What to do |
| --- | --- |
| Write a blog post | Click **Preview** in the toolbar, then type `localhost:4321/keystatic` in the address bar |
| Publish changes | Type `/deploy` |
| Customize the design | Type `/design-interview` |
| Check the site for problems | Type `/check` |
| Fix something | Type `/fix` |
| See the live site | Visit https://pairadocs.farm |

## Weekly workflow

1. Open this folder in Claude Desktop's Code tab
2. Click **Preview** to start the site
3. Navigate to `localhost:4321/keystatic` in the preview
4. Fill in the form (title, description, date, image, tags)
5. Write your post, click **Save**
6. Type `/deploy` to publish — live in about a minute!
7. Share on social media (Facebook, Instagram, Nextdoor)
8. Go back to Keystatic, add share URLs to **Syndication Links**, save, and `/deploy` again

## Getting help

Type anything in the chat — describe what you need in plain English. Your webmaster can write new pages, fix problems, update the design, draft emails, and more.

## Where things live

- **Your website project:** iCloud Drive → Pairadocs Farm
- **Live site:** https://pairadocs.farm
- **Analytics:** Cloudflare Web Analytics
- **Members:** Airtable (set up with `/setup-airtable`)
