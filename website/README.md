# Pairadocs Farm 🌱

This is the source code for your farm website. It lives in iCloud Drive so it's backed up automatically.

## Your app

**Pairadocs Farm** is in your Applications folder and (after setup) in your Dock. Click it to see the task menu:

| I want to… | Click… |
|---|---|
| Write a blog post | **✏️ Edit Posts** |
| Publish changes | **📤 Publish to Farm** |
| See visitor stats | **📊 View Analytics** |
| Manage CSA members | **🗂️ Open Airtable** |
| See the live site | **🌐 View Live Website** |
| Ask for help | **🤖 Chat with Webmaster** |

## Weekly workflow

1. Click **Pairadocs Farm** → **✏️ Edit Posts**
2. Fill in the form (title, description, date, image, tags)
3. Write your post, click **Save**
4. Click **Pairadocs Farm** → **📤 Publish to Farm**
5. Live in about a minute!
6. Share on social media (Facebook, Instagram, Nextdoor)
7. Copy share URLs → Edit Posts → add to **Syndication Links** → Save → Publish again

## Getting help

Click **Pairadocs Farm** → **🤖 Chat with Webmaster** and describe what you need in plain English. Your webmaster can write new pages, fix problems, update the design, draft emails, and more.

## Where things live

- **Your website project:** iCloud Drive → Pairadocs Farm
- **The app:** Applications → Pairadocs Farm
- **Live site:** https://www.pairadocs.farm
- **Analytics:** Cloudflare Web Analytics (via the app menu)
- **Members:** Airtable (via the app menu)
