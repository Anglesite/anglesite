Anglesite JSON Schemas

JSON Schema definitions for Anglesite website configuration files.

Main Schema

  • website.schema.json
    Main schema for Anglesite website configuration
    Use in your website.json: "$schema": "https://anglesite.dwk.io/schemas/website.schema.json"

Module Schemas

  • basic-info.json
    Basic website information (title, description, URL, etc.)
  • seo-robots.json
    SEO and robots.txt configuration
  • web-standards.json
    Web standards (favicon, manifest, etc.)
  • networking.json
    Networking and server configuration (headers, redirects)
  • analytics.json
    Analytics configuration
  • well-known.json
    .well-known standards configuration
  • common.json
    Common schema definitions and types

Usage

To use these schemas in your VS Code or other JSON editors, add the $schema property to the top of your JSON files: 

{
  "$schema": "https://anglesite.dwk.io/schemas/website.schema.json",
  "title": "My Website",
  ...
}
